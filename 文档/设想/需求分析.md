# 1. 功能

## 1.1 基本功能

前提：甲乙都安装了app

- 1.1.1 甲乙相距很近，甲发帖，乙收到；反之亦然
- 1.1.2 甲乙相距很远（大于蓝牙理论距离如 300米），甲发帖，乙靠近后收到；反之亦然

## 1.2 其他

- 1.2.1 帖子长度无限制

# 2. 界面

## 1.3 极简

- 2.1.1 无帖子时间戳
- 2.1.2 无发帖人信息
- 2.1.3 默认设置为收到新帖完全不通知
- 2.1.4 与其他peer同步帖子不显示进度百分比
- 2.1.6 无删帖功能
- 2.1.7 无搜索功能
- 2.1.8 无屏蔽关键词
- 2.1.9 收费版和免费版的功能和界面完全相同（如store支持30天之内可重装，则只需提供收费版）
- 2.1.10 无连接状态显示

## 2.2 帖子列表
 
- 2.1.1 所有帖子向右对齐（一般聊天软件中，自己发的消息都在右边）
- 2.1.2 最新的帖子在最上面
- 2.1.3 完全相同的帖子发出后，旧帖消失，新帖在最上面
- 2.1.4 支持复制选中的帖子内容
- 2.1.5 每帖都完整显示，不管多长

## 2.2 文本输入

- 2.2.1 仅支持纯文字内容，不支持图片、视频、链接等
- 2.2.2 支持粘贴内容

## 2.3 『记』按键

- 2.3.1 文本输入为空时disable；反之enable
